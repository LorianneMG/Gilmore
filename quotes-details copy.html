"Quotionary" (Use LocalStorage)

Project information

  • Description: Custom Image Gallery
  • Category: Web design
  • Github URL: www.example.com
  • Used: PHP | BOOTSTRAP | HTML | CSS

The Details

Highly-stylized personal image gallery using PHP. Upload input shows file name.

Hover over image - slightly zooms in, darkens overlay with a delete button that has a glow rotating around it for flair.

Flex box to wrap images. Images also change size to always keep rows at the appropriate size.
